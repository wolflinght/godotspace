# 《星河放置：暗面危机》任务、状态与资源生态策划案（V2.0）

- **文档编号**：SYS-007  
- **系统定位**：状态机运转逻辑、全服资源分配、动态航行结算  
- **核心原则**：动态结算、有限资源池、风险与目标驱动  

## 一、玩家状态机与动态耗电机制（State Machine & Dynamic Power）

游戏底层采用严格的三态互斥机制。

### 状态 1：【停泊】（Docked）

- **状态特征**：飞船停靠在具体的 POI 上，处于静止和绝对安全状态。  
- **空间站补给规则（修改）**：空间站不再提供一键瞬间满电，而是提供一条免费的「充电管道」。  
- **规则**：玩家点击「连接电网」后，飞船以 **10 电力 / 分钟** 的速率缓慢充能，直至达到当前电池上限。  
- **策略**：要求玩家合理规划下线时间。白天急需用电时，只能花星币买「燃料」用反应堆瞬间发电；晚上睡觉时把船停在空间站白嫖电网。  

### 状态 2：【航行中】（In-Transit）

- **状态特征**：飞船正在星际空间中移动，锁定所有装配与交易功能。  

#### 动态耗电系统（Dynamic Power Drain）

- **规则**：航行不再在起飞时一次性扣除全部电力，而是伴随飞行时间逐分钟动态扣除。
- **公式**：每分钟耗电率 = 单次航行总预计耗电 / 预计航行总分钟数（结果四舍五入到整数；最后一分钟按实际剩余耗电结算，消除累计误差）。  
- **极端情况（抛锚）**：若起飞前未准备足够燃料/电力，或航行中被怪物攻击导致引擎漏电，一旦电力在航行途中动态归零，飞船进入「太空抛锚」状态。  
- **抛锚惩罚**：速度强制降为 1（靠惯性漂流），航行时间被无限拉长，必须等待“其他玩家到达本位置营救（到达后可以输送电力）”或呼叫天价星际救援。  

### 状态 3：【作业中】（Working）

- **状态特征**：抵达目标地并展开作业。根据 POI 不同，执行采矿、打捞等固定挂机动作。  
- **表现**：界面显示当前收益流水与资源获取倒计时；玩家随时可点击「中止作业并返航」。  

## 二、任务委托系统（Mission Board System）

除自由挂机外，玩家可在各阵营空间站 / 自由港接取官方委托任务。任务系统是玩家获取星币和声望的主要来源。

**任务时限规则**：所有任务的时限从**接取任务时**开始计时，而非起飞时。玩家需自行规划航行+作业时间，确保在时限内完成并交付。

### 1）派送 / 运输任务（Delivery）

- **目标**：将系统发放的特殊货物（如：高级军火、机密数据盒）从 A 空间站运送到 B 空间站。
- **特征**：适合新手与低战力玩家，考验飞船「速度」与「电池续航」。
- **限制**：货物占用飞船「舰仓」容量，压缩高级武器装配空间；中途被打爆则任务失败并扣除巨额违约金。

### 2）采矿定单（Mining Request）

- **目标**：在规定时间内前往指定 POI 采集 X 吨特定矿物，并交还给发布任务的空间站。
- **特征**：相比直接卖矿给黑市，完成定单可获得额外 **30%** 星币奖励与阵营声望。

### 3）悬赏讨伐（Subjugation / Bounty）

- **目标**：前往指定危险航线或废墟 POI，击杀特定数量机械丧尸或某个具名海盗头目。
- **特征**：纯粹战力检测，通常奖励稀有组件图纸碎片或特殊战利品箱。
- **击杀计数规则**：单次遭遇战中击杀多只目标敌人，全部计入击杀数。可在多个 POI 累计。

## 三、POI 资源池与每日枯竭机制（POI Resource Reserves）

为模拟真实的宇宙开发生态，星图上的资源不是无限的。**资源池与危险度不在本系统单独维护**，统一以星图 POI 配置为准（见 `SYS-005 map.md` 各星系「节点大全表」中的“资源池默认值”“危险度”字段）。

### 1）资源储量定义（POI 配置项）

- 每个非空间站 POI（如行星、废墟、资源带）拥有独立的资源池（可按联机架构选择全服共享 / 小队 / 个人）。  
- 推荐最小字段：`resource_pool_defaults`（资源类型与每日储量上限/默认值）、`resource_pool_remaining`（当前剩余量）。  

### 2）采矿与枯竭逻辑

- 玩家进入「作业中」状态后，每小时产出从该 POI 的「当前剩余量」中扣除。  
- **枯竭判定**：当储量被挖到 0 时，POI 进入 **枯竭（Depleted）** 状态；继续挂机则矿机产出变为 0（UI 弹红字：本星区资源已枯竭）。  

### 3）天级刷新机制（Daily Reset）

- 现实时间每天 00:00（或服务器设定的天运转周期），宇宙引力潮汐爆发或陨石雨降临。  
- 所有 POI 资源储量瞬间恢复至 100% 上限。  
- **设计目的**：避免玩家在绝对安全的「新手村卫星」挂机一辈子；安全区挖空后，玩家为不浪费挂机时间被迫升级护盾与引擎，前往更遥远、更危险的深空开采富矿星。  

## 四、航行遭遇战机制规范（Encounter System）

结合动态航行，危险遭遇系统将更真实。

### 1）危险度评级（Danger Level，来自 POI 配置）

星图不同区域的基础危险度由 0 到 5 级递增，**具体数值以 POI 配置为准**（见 `SYS-005 map.md` 各星系「节点大全表」的“危险度”字段）。  

### 2）遭遇 Roll 点逻辑

- **判定窗口（按距离分段）**：航行中每飞行 **10 个距离单位**触发一次遭遇判定。例如铁砧-IV → 17号船坟（距离28）途中触发 2 次判定。短途（距离 < 10）也有机会在抵达时触发 1 次。
- **遭遇概率（按目标POI危险度）**：

| 危险度 | 遭遇概率 |
|--------|----------|
| 0 | 5% |
| 1 | 10% |
| 2 | 15% |
| 3 | 20% |
| 4 | 25% |
| 5 | 30% |

- **高风险高收益**：遭遇敌人级别严格与区域危险度挂钩。Level 4 区域遭遇的机械丧尸，其爆出的星币、战利品基础量是 Level 1 区域的 **4–5 倍**。  

### 3）战斗插入处理

- 触发遭遇后，航行动力系统暂停（耗电暂停、倒计时冻结）。  
- 后台瞬间调用战斗公式（SYS-006）进行全自动演算。  
- 玩家胜利：获得战利品，航行动力恢复，继续前往目标点。  
- 玩家战败 / 爆船：航行强制终止。飞船扣减战损 HP，强行弹射回最近的「空间站」，已消耗电力不退还。  

## 五、玩家一日游戏切片推演（Day-Loop Simulation）

- **[08:00] 基地谋划**：玩家在「铁砧-IV」空间站上线。昨晚接通电网，现在飞船电力全满。查看任务看板，接取一个「送货」任务和「塔洛斯星采矿定单」。  
- **[08:10] 动态出航**：玩家装配高机动引擎，启航前往塔洛斯。航程预计 40 分钟，总耗电 160；随时间流逝，电力每分钟下降 4 点。  
- **[08:30] 遭遇拦截**：航行到一半触发 Level 2 遭遇战。一伙海盗袭来。战报显示：护盾破裂，本体掉血 20 点，但反杀海盗获得 500 星币。  
- **[08:50] 抵达与作业**：抵达塔洛斯星，状态切为「作业中」。玩家下线工作。  
- **[14:00] 资源枯竭**：玩家上线查看。定单要求的 5000 吨钛合金早已挖够；但因当前星球资源储量只有 10000 吨，此时星球已「枯竭」，不再产出。  
- **[14:10] 极限返航**：玩家点击返航交任务。剩余电力不足以飞回基地，花星币买一罐燃料，用反应堆瞬间转化补足电力后起飞。  
- **[15:00] 交差与休整**：回到空间站交付定单大赚一笔。修好装甲，将飞船重新连上电网，然后挂机等待每天晚上的全服丧尸大防守。
