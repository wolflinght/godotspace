# 《星河放置：暗面危机》星图矩阵与节点生态策划案（V3.0）

- **文档编号**：SYS-005  
- **系统定位**：挂机目标分布、地图节点定义、势力领地划分、航线耗电数据源  

## 一、星系节点设计准则（Node Design Rules）

- **多阵营枢纽**：每个星系不再只有一个孤立的空间站，而是由三大基石构成：「铁骑财团前哨」「光斑序列枢纽」「中立黑市 / 自由港」。玩家可以自由选择停泊点（起点）来规划航线，以节省耗电。
- **节点丰富度**：星系内部包含各类天体（类地行星、气态巨行星、卫星、小行星带）和废墟人工物，构成 10–15 个可选任务节点，产出不同偏向的资源。
- **距离与地缘**：阵营的空间站通常靠近它们控制的富矿行星；而危险的恒星和丧尸废墟通常位于星系边缘。
- **光照强度**：每个 POI 拥有光照强度属性（极高 / 高 / 中 / 低 / 绝地），直接影响飞船太阳能板组件的发电效率。光照强度与发电倍率对应关系：极高=200%、高=150%、中=100%（基准）、低=50%、绝地=0%。

## 一-B、跨星系跃迁系统（Inter-System Jump）

- **解锁条件**：完成当前星系的「终极任务」后，玩家获得**跃迁资格**，并可在空间站购买「跃迁引擎」组件（安装于舰尾槽位）。
- **跃迁规则**：
  - 玩家可从当前星系任意位置发起跃迁，抵达目标星系的**指定跃迁锚点**（通常为该星系的中立空间站）。
  - 跃迁消耗固定电力（具体数值由跃迁引擎组件的 P.Cost 决定），不受距离矩阵影响。
  - 跃迁期间进入独立的「跃迁中」状态，不触发遭遇战，不可中断。
- **跃迁引擎**：作为特殊组件存在，安装后才能使用跃迁功能；未安装则无法跨星系移动。  
## 二、宇宙四大阵营与「失落帝国」设定

### 1）【奇点圣约】（Singularity Covenant）——「太虚监察者」

- **势力特征（失落帝国 / 唯心主义）**：这是一个极其古老、科技水平呈现绝对碾压态势的失落文明。他们不追求扩张，也不在乎凡俗的矿物资源。他们信仰「宇宙的深层精神秩序（灵能与高维意识）」，认为物理宇宙只是一场修行。  
- **阵营态度（太空警察）**：极度傲慢且不可理喻的「宇宙执法者」。他们平时对凡人势力的厮杀视而不见，但若某些行为触碰「宇宙禁忌」（如：光斑序列试图引爆超新星、铁骑过度开采导致重力井崩塌，或月之暗面感染到达临界点），高维舰队会突然降临进行无差别降维打击，强制恢复平衡。  
- **组件特征**：玩家极难直接购买到 [奇点] 前缀装备；多为远古战争遗留在黑洞边缘的「圣遗物」。  

### 2）【铁骑财团】（Ironclad Syndicate）——「物理压榨者」

- **势力特征**：宇宙最大的实体重工巨头。极端唯物，奉行大炮巨舰与暴力开采。极其贪婪，是当前宇宙乱局的主要制造者。  

### 3）【光斑序列】（Macula Sequence）——「狂热进化派」

- **势力特征**：由极端科学家组成的寡头。技术极度先进（热能 / 护盾），道德极度低下。他们试图捕获并解析「月之暗面」病毒，期望借此让人类实现机械飞升，极其危险。  

### 4）【月之暗面】（The Dark Side）——「机械瘟疫」

- **势力特征**：一种能感染底层代码并强制重组废弃星舰、矿机的机械丧尸病毒。它们没有高等智慧，只有同化一切金属与能源的本能，是遍布各大星系废墟地带的常规毒瘤。
